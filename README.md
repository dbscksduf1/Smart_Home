# 🏠 Smart Home Environment Monitoring System

실내 환경 센서 데이터를 기반으로  
현재 공간의 상태를 시각적으로 확인하고  
AI 분석을 통해 관리 조언을 제공하는 스마트 홈 모니터링 시스템입니다.

---

## 📌 프로젝트 개요

- 실내 환경 정보(온도, 습도, 조도, 가스, 소음, 공기질)를 수집
- 센서 데이터를 직관적인 대시보드 형태로 시각화
- 환경 상태를 종합 분석하여 AI 기반 관리 조언 제공
- 실제 센서가 없는 환경을 고려해 시뮬레이터 기반으로 설계

---

## 🔄 센서 데이터 시뮬레이션 방식

실제 하드웨어 센서가 없는 환경을 고려하여  
Spring Boot 내부에 센서 시뮬레이터를 구현했습니다.

- 1분 주기로 랜덤 센서 값 생성
- 온도, 습도, 조도, 가스, 소음, 공기질 데이터 포함
- 최신 데이터는 `/sensor/latest` API를 통해 제공

```text
SensorSimulator (@Scheduled)
 → 최신 센서 데이터 생성
 → 전역 저장
 → 프론트엔드에서 조회
🤖 AI 환경 분석 기능
센서 데이터를 기반으로 현재 실내 환경 상태를 분석하고
사용자가 이해하기 쉬운 형태의 조언을 제공합니다.
